##  📝 프로젝트 소개

> 코로나 시대에도 안전하게❗ Psycho와 함께라면 어디든 갈 수 있다❗



**Psycho**는 코로나 시대에 다수의 유동인구를 밀집지역으로부터 회피시키고 불가피한 이동시 안전한 곳으로 분산시키는 플랫폼 입니다.

요즘 트렌드에 가장 적합한 코로나와 언택트시대에 초첨을 맞춰서 생각해보았습니다. 코로나가 확산되면서 많은 사람들은 불안해했습니다. 그럼에도 불구하고 다수의 유동인구가 있었으며 밀집지역을 회피하고자하는 경향이 커졌습니다. 결국 불가피한 이동이 발생할 수 밖에 없습니다. 그래서 사람들이 코로나의 위험을 최대한 피할 수 있도록 선택지를 제공하자는 생각에 사람들을 안전한 곳으로 분산시키는 서비스를 기획했습니다.

**Psycho**는 코로나 관련 뉴스와 SNS에서 코로나 관련 키워드를 크롤링하여 게시하고 각종 코로나 관련 지표들을 시각화 하여 메인페이지에 게시함으로써 사용자가 코로나를 항상 자각하도록 유도 하였습니다. 그리고 카드사 데이터와 지역별 코로나 확진자 현황 등 공공데이터를 활용하여 안전한 장소와 시간대에 음식점 및 숙박업소를 추천할 수 있도록 구현하였습니다.

## 🔧 담당 역할 (구현 사항)

- 네이버에 게시되는 언론사별 코로나 관련 뉴스 크롤링
- SNS 크롤링 및 형태소 분석을 통해 키워드를 추출, 통계 및 시각화
- 공공데이터포털 API 데이터셋을 가공 후 시각화
